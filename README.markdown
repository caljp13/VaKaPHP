VaKa PHP Framework
=============
------------------

This is a PHP framework compact and functional! 
Based on [Toro](http://www.toroweb.org) and [MicroMVC](http://www.micromvc.com)


### Requirements

* PHP 5.3+
* Apache mod_rewrite (or similar for other servers)
